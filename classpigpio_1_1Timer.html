Detailed Description

Simple timer that can be used to execute instructions at a given rate.

This class is similar to Rate, the main difference being that it does not blocks the main thread by sleeping. It is meant for executing instructions at a rate that is lower than that of the control loop.

◆ expired()

bool pigpio::Timer::expired ( )

Tells if the timer expired.

This method checks if the timer has expired. If auto_reset=true was passed to the constructor, a call to reset() is also performed. Otherwise, the timer is not reset, meaning that further calls to expired() will keep returning true. In this case, the timer has to be reset manually by a call to reset().

Returns
true if the timer expired.
