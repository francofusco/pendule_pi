◆ readState()

bool pendule_pi::PenduleCpp::readState ( bool  blocking)

Tries to read the state of the pendulum from the interface.

Parameters
blockingif true, do not exit until a message has been received from the socket. If false, exit immediately if no messages are received.
Returns
true if a message has been received and processed. Note that if the call is blocking, the function should always return true.
Note
To make the code more readable, the members BLOCKING and NON_BLOCKING have been defined. You are encouraged to call this method as, e.g.:
{.c++}
PenduleCpp pendulum;
pendulum.readState(PenduleCpp.BLOCKING);
pendulum.readState(PenduleCpp.NON_BLOCKING);
